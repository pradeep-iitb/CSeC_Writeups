## Rules & Guidelines

1. **No brute-force attacks** on live challenge infrastructure.
2. **Do not share flags** with other participants.
3. **Document your approach** — understanding *how* is more valuable than the flag itself.
4. If you're stuck, re-read the Readme carefully — every word is intentional.
5. Have fun and learn something new! 🚀
